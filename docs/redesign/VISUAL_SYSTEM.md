# Visual system

Museion uses a warm neutral canvas, white working surfaces, deep ink, lapis actions, restrained gold for attention, and semantic green/red only for verified states. Typography uses the native system sans stack so no font is claimed without being loaded.

## Tokens

- Canvas: `paper` (`#f7f6f2`)
- Surface: `surface` (`#fff`)
- Primary text: `ink`
- Secondary text: `ink-soft`
- Primary action: `lapis`
- Attention: `gold`
- Card radius: `--radius-card` (`1rem`)
- Control radius: `--radius-control` (`.75rem`)
- Default elevation: `--shadow-tight`

## Composition rules

- Prefer compact modules over one giant “showcase” card.
- Use labels in sentence case; reserve monospace for values and technical provenance.
- One dominant action per module.
- Percentages represent only direct completion ratios, never mastery confidence.
- Evidence state and recommended support are separate fields.
- Decorative motion must yield to reduced motion and never imply live processing.
