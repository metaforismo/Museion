# Design system V2

Museion V2 is editorial, calm, and instrument-like: paper background, white working surfaces, lapis for decisions, gold for attention, green/red only for checked outcomes. It avoids gradient-led AI styling, inflated cards, gamified urgency, and decorative dashboards.

- Typography: system sans for honest performance; tight display tracking, readable body line-height, monospaced micro-labels only for evidence/protocol metadata.
- Radius: 8–12 px for controls, 20–26 px for primary surfaces, never nested indiscriminately.
- Shadows: one restrained elevation token; borders do most grouping.
- Density: compact headers, high-information rails, progressive disclosure for rationale and status.
- Motion: state-change only; no continuous mascot animation; reduced-motion remains respected.
- Accessibility: 44 px actions, visible focus, semantic landmarks, status regions, non-forcing chat scroll, keyboard-complete navigation.
