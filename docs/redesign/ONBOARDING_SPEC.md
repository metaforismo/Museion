# Onboarding specification

Implemented: five keyboard-accessible steps covering Museion, deterministic checking, Maia, adaptive support, and optional personalization. Learners may go back, skip, restart by clearing local state, or select role, broad age band, and first goal. Preferences are versioned and stored only in the browser; malformed storage falls back safely. No exact birth date, school, or public profile is requested.

Planned before youth deployment: supervised mode, parent/educator information and consent flow, live-AI gate, one-click product-level deletion, language/subject selection, save-and-resume across devices, and an explicit privacy/model step.
