# Dashboard V2

The primary object is a compact, server-ranked mission—not a dark promotional banner. It contains the next justified learning move, a disclosed rationale, and one purposeful Maia state. Review and evidence counts sit beside it; the Museion protocol forms a dense orientation rail below.

Empty states recommend real authored lessons or source creation and do not invent streaks, XP, mastery, or urgency. Runtime badges now state that authored learning records are local even when generated work can use durable storage. Evidence appears only after a checked response.
