# Course authoring

One workstream owns one subject, learner band, source set, content directory, fixtures, misconception tests and evidence notes. Authors separate source facts, generated pedagogy, private evaluator truth and learner-visible content.

Release requires span/hash checks, answer privacy, deterministic answer/rubric truth, runtime compatibility, one critic and at most one typed repair, fail-closed publication, accessibility notes and a transfer task without Maia/hints/solutions. External standards require provenance and license review; generated content is never silently official curriculum.
