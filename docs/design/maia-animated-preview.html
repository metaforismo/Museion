Museion · brand · Maia

Maia, recreated and set in motion

The five poses of the Maia character sheet, traced pixel-for-pixel into vector paths and animated with CSS that lives inside each SVG file. Everything on this page is self-contained: the mascots above are the production files, embedded as data URIs.

What each pose does

neutral

Breathes gently; blinks on a 4.6-second cycle.

public/brand/maia/animated/maia-neutral.svg

thinking

The question mark bobs and tilts while the eyes glance up and blink.

public/brand/maia/animated/maia-thinking.svg

celebrating

Hops every 1.7 seconds as the three sparks pop in a staggered burst.

public/brand/maia/animated/maia-celebrating.svg

curious

Thought bubbles drift upward one after another between blinks.

public/brand/maia/animated/maia-curious.svg

pointing

Breathes and blinks, arm held out mid-gesture.

public/brand/maia/animated/maia-pointing.svg

Status size

Motion & accessibility

All animation sits inside a prefers-reduced-motion: no-preference media query in the SVGs themselves — if your system asks for reduced motion, every pose freezes to its static frame automatically. The static frame is pixel-identical to the static family in public/brand/maia/.
